* Orientação ao Objeto: { 
Estudar orientação ao objeto com JavaScript, porque a linguagem também é orientada ao objeto.

- Objetos é mais evoluido do que um array, porque no array só posso acessar os elementos com valores numericos, mas no objetos, é possivel acessar os elementos atraves de string e até chamar functions no objeto.

- Objetos são variaveis, a variavel guarda valores, só que alem de guarda valores que são os atributos, os obejtos podem guarda funcionalidade (functions) que é chamado de métodos

No JavaScript o array é um objeto e o objeto é um objeto, porque tem a estrutura da mesma origem.
